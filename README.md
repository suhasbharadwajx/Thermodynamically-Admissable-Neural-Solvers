# Thermodynamically Admissible Neural Solvers for Stiff Electro-Elastodynamics

Implementation of the constrained Physics-Informed Neural Network (PINN) for 1D coupled electro-elastodynamics, as described in the manuscript: *"Thermodynamically Admissible Neural Solvers for Stiff Electro-Elastodynamics via Exact Geometric Constraints."*

## Usage
To reproduce the results:
1. Install requirements: `pip install -r requirements.txt`
2. Run training: `python src/train.py`
